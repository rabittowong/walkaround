    What is WalkARound

    WalkARound is a iOS application using augmented reality (AR) and location-based technologies
    for promoting self-motivated learning. By making the attractions information easily accessible
    through an interactive AR interface, WalkARound helps you navigate effortlessly.

    Join us now to become an active learner!

    WalkARound will guide you through
    community-created paths, which
    are displayed in AR.
